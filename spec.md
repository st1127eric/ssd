# 貪食蛇遊戲規格文件

## 專案概述
一個使用 HTML、CSS 和 JavaScript 開發的經典貪食蛇遊戲，可部署在 GitHub Pages 上。

## 功能需求

### 核心功能
1. **遊戲場地**
   - 固定大小的方格遊戲區域（建議 20x20 格）
   - 清楚的邊界顯示

2. **蛇的移動**
   - 蛇由多個方格組成
   - 初始長度為 3 格
   - 使用方向鍵控制移動（上、下、左、右）
   - 蛇持續朝當前方向移動

3. **食物機制**
   - 隨機位置生成食物
   - 食物不會出現在蛇身上
   - 吃到食物後蛇身長度增加 1 格
   - 吃到食物後生成新的食物

4. **遊戲結束條件**
   - 撞到牆壁
   - 撞到自己的身體

5. **計分系統**
   - 顯示當前分數
   - 每吃一個食物加 10 分
   - 顯示最高分數（使用 localStorage 保存）

6. **遊戲控制**
   - 開始/重新開始按鈕
   - 暫停/繼續功能

### 使用者介面
- 遊戲標題
- 遊戲畫布（Canvas）
- 分數顯示區
- 遊戲說明
- 控制按鈕

## 技術規格

### 技術棧
- HTML5（Canvas API）
- CSS3
- Vanilla JavaScript（無框架）

### 檔案結構
```
/
├── index.html
├── style.css
├── game.js
└── README.md
```

### 遊戲參數
- 格子大小：20px
- 遊戲區域：20x20 格（400x400px）
- 遊戲速度：150ms/格（可調整）
- 初始蛇長：3 格
- 初始方向：向右
